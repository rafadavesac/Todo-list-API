# Todo-list-API

Um projeto fullstack de lista de tarefas que desenvolvi do zero. O backend é uma REST API feita com Node.js e Express, com rotas para criar, listar, editar e deletar tarefas. O frontend foi construído com HTML, CSS e JavaScript puro, se comunicando com a API através do `fetch` e atualizando a página dinamicamente sem precisar recarregá-la.
